Data Scientist Code of Ethics

This code aims to define values and priorities for overall ethical behavior, in order to guide a data scientist in being a thoughtful, responsible agent of change.

OVERVIEW

These principles are created and maintained by the data science community for guiding anyone that uses data to affect the lives of individuals to do so responsibly. It is not intended to be all encompassing, rather with these principles, a common foundation is created for standards of responsible data use in academia, industry, for specific sectors and for individuals.

Our working groups centered around seven topics:

Thought Diversity Privacy and Security Responsible Communication Questions & Answers Provenance and Ownership Transparency Bias
